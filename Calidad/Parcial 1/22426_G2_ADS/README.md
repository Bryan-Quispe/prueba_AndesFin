# Sistema de Gestión de Inventario y Órdenes de Trabajo - PintAuto

Este repositorio contiene el desarrollo del sistema PintAuto. Se incluyen artefactos de análisis, diseño, desarrollo y pruebas.

## Estructura
- `/Documentacion`: Requisitos, diseño, pruebas y gestión del proyecto.
- `/CodigoFuente`: Backend en Spring Boot y frontend en React.
- `/Releases`: Versiones liberadas del software.
- `/Backups`: Copias de seguridad.
