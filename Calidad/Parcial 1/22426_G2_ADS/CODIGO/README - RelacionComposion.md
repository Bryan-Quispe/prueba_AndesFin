## Relación entre Código y Diagrama de Componentes

Este apartado describe la relación existente entre el código desarrollado y el diagrama de componentes elaborado para el backend y frontend del proyecto.

Contar con esta relación nos permite identificar de forma clara cómo se han aplicado los cambios en nuestro código, facilitando la trazabilidad y validación del desarrollo.
